# Contribute to SerialPilot Bindings C++

Thanks for helping improve `@serialpilot/bindings-cpp`.

## Reporting Issues

- Search existing issues first: https://github.com/serialpilot/bindings-cpp/issues
- Open a new issue with repro steps, expected behavior, and actual behavior.

## Requesting Features

- Open a feature request at https://github.com/serialpilot/bindings-cpp/issues
- Include the use case, desired behavior, and why it matters.

## Submitting Pull Requests

1. Fork this repository and create a branch.
1. Implement your change with tests.
1. Run `npm run lint` and `npm test`.
1. Open a pull request to this repository and describe the change clearly.

All code contributions should follow the existing lint rules and coding style.

## Writing Tests

Tests use mocha, chai, and sinon. Include tests for every bug fix or feature.

## Writing Documentation

If docs are missing or unclear, submit improvements in your pull request.
