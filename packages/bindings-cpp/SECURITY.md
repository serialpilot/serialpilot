# Security Policy

## Supported Versions

We currently support the latest version of `@serialpilot/bindings-cpp` only.

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

Please email contact@riteshrana.engineer if you find any issues and we'll coordinate disclosure.
