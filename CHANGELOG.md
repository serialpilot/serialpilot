# Change Log

All notable changes to this project will be documented in this file.

## 1.0.0 (2026-05-08)

### Features

- Initial public release of SerialPilot v1.0.0 under the `@serialpilot` npm scope.
- All packages, classes, and CLI binaries are published under the `@serialpilot/*` and `serialpilot-*` namespaces.
- Native bindings (`@serialpilot/bindings-cpp`) and pure-Rust crate (`serialpilot-rust`) ship from the same monorepo.
